Basics

First argument is an object that maps CSS properties to their desired final values.

Second argument is an object that specifys animation options.

Shorthand: comma-seperated argument syntax instead of passing in second object.

      duration (integer), easing (string), complete (function value)

If you want any other animation options, you must use object syntax.

There are 2 differences between velocity annimation and CSS annimation:

  1. Velocity only accepts a single numeric value per CSS property.
  2. 'padding-left' becomes 'paddingLeft'...
